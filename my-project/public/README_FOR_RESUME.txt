To include your PDF resume on the site:

1. Copy your existing `Resume G.pdf` into this folder and rename it to `resume.pdf`.
   - Source (example): c:\gauthami_portfolio\Resume G.pdf
   - Destination: c:\gauthami_portfolio\my-project\public\resume.pdf

2. Start the dev server with `npm run dev` and the "Download Resume" button will open the PDF.
